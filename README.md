# RMD-TVReg

**RMD-TVReg** (Robust and Missing-Data-Aware Time-Varying Regression) is a framework for estimating high-dimensional regression models with time-varying coefficients in the presence of outliers and missing data.

The method integrates:
- robust loss functions,
- EM-based handling of missing values,
- temporal smoothness regularization, and
- nonconvex penalties for sparsity,

to produce stable and interpretable coefficient trajectories.  
RMD-TVReg is designed for dynamic real-world datasets where robustness and structural interpretability are more important than short-term predictive accuracy.
